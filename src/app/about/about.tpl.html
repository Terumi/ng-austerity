ng-Austerity For the lazy and impatient.

ng-austerity is a web version of the solitaire boardgame Austerity, designed by Jake Staines .

Why?

This small game is awesome, I really liked playing it in its physical form and wanted to be able to take it allong with me. This webversion should play nicely on a small screen like a mobile device.

Can I Help?

Yes, please!

If you are a AngularJS developer and want to join in, feel free to fork the code and send pull requests.

The main thing needed is testing, find bug and file issues ! And tell me how friendly (or not) the UI is.
